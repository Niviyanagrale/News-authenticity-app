--News Authenticity Detection App--

Built with Python • scikit-learn • Streamlit

This project is a machine learning–based application that classifies news articles as Real or "Fake" 
using Natural Language Processing techniques.

The project was created as a learning project to explore text classification
and to implement regression-based machine learning models and NLP preprocessing techniques that 
were taught to us in the second semester- Aiml python 
we learnt it on jupyter Notebook

Purpose of the Project

This application was built to experiment with and understand 
how machine learning models can be applied to real-world problems such as misinformation detection. 
It was primarily developed as a 
practice project for applying AI/ML concepts including vectorization, regression models, and model deployment.

-- Tech Stack
Python
scikit-learn
pandas
NumPy
Streamlit

--Machine Learning Concepts Used
- Text preprocessing and cleaning
- TF-IDF Vectorization
-Logistic Regression for classification
- Model training and evaluation

Features
 Classifies authenticity of news texts based on its wording, paraphraasing and other components
 Interactive user interface built with Streamlit
 Uses a trained ML model for predictions
